# YouTube Summary Telegram Bot

Send a YouTube link to this bot and it will reply with a 4-bullet summary.

## Setup

1. Create Telegram bot and get token
2. Get Groq API key
3. Set environment variables:
   - TELEGRAM_BOT_TOKEN
   - GROQ_API_KEY
4. Install dependencies:
   pip install -r requirements.txt
5. Run bot:
   python bot.py
